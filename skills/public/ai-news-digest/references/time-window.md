# 时间窗口解析规范

目标：允许用户用自然语言或日期字符串指定“抓取哪一天的资讯”，并明确时区解释规则。

## 默认行为

- 未提供时间相关输入时，默认等同于：`今天`（用户时区的当天 00:00-23:59:59）
- 未提供时区时，默认 `UTC+8`（建议实现默认 `Asia/Shanghai` 并在输出中显式写明）

## 支持的输入（建议）

- 中文自然语言：`今天` / `昨天` / `前天`
- 英文自然语言：`today` / `yesterday`
- 指定日期：`YYYY-MM-DD`
- 高级（可选）：`最近24小时` / `过去7天`（如实现，需在输出里标注窗口）

## 时区规则

- 以用户指定 `--tz` 为准；未指定时用本机时区（需在输出中显式写明时区）。
- “YYYY-MM-DD”解释为该时区的自然日窗口。

## 输出约定

- 解析结果必须输出 `(since, until)`，均为带时区的 ISO 8601。
